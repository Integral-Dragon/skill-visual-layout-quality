# Future Enhancements

This folder is the holding area for post-v1 ideas.

Use it to capture enhancements before they are committed to the core skill.

The point is to preserve:

- who suggested the idea
- what the enhancement is
- why it matters
- how expensive or uncertain it looks

## Subfolders

- `Dex/`
- `Hopper/`
- `Max/`

## Scoring

Use a `1-5` scale for each dimension.

- `Effort`
  - `1` = trivial
  - `3` = moderate
  - `5` = large / multi-step
- `Uncertainty`
  - `1` = straightforward / low ambiguity
  - `3` = some open technical or workflow questions
  - `5` = highly uncertain or likely to need experimentation
- `Impact`
  - `1` = minor polish
  - `3` = useful improvement
  - `5` = major improvement to the skill's practical value

These are intentionally rough planning signals, not formal estimates.
